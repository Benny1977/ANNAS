Samanage Dashboard

The dashboard is a tool for both managers and agents to view key metrics on current work
The purpose of redesigning it was to make it more customizable and user friendly, as well as to allow sharing customized dashboards between users
Also, there was a need to add an optimized view for TV

TV Mode

Each dashboard can be shown in TV Mode. In the TV Mode all widgets are enlarged and the header panel is hidden
